Sobre alguns conceitos importantes:

    DIV:

  • A tag "div"> é um elemento HTML que serve como um contêiner genérico, podendo receber diferentes tipos de conteúdos (texto, imagens etc, elemento bastante utilizado para a construção de layouts de páginas web e pode facilmente ser estilizado com o uso do CSS.

    • Diretiva HTML:

  • Elas aprimoram a capacidade dos elementos do HTML anexando comportamentos personalizados. Exemplo: height,width,align...

    • O que é Bootstrap:

  • Bootstrap é um framework front-end que fornece estruturas de CSS para a criação de sites e aplicações responsivas de forma rápida e simples. Além disso, pode lidar com sites de desktop e páginas de dispositivos móveis da mesma forma.
